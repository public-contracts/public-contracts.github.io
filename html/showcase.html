It's Public & Decentralized

  • The network uses only P2P interaction without discrimination
  • All digital contracts are public and are distributed throughout the network

  • The network uses only safe and economical connections

It's Easy

  • The network uses the most popular Internet standards
  • Community provides all the necessary software

Network features

Public vote

Public decentralized vote in which everyone can create, accept or discard petitions.

Worldwide payments

Public decentralized virtual coin transfers that anyone can make by performing computational work.

Digital Contracts

Public decentralized distribution and digital signature of contracts with end time.

Anonymous search

Querying with the identifier search for a response from the desired person and one-on-one interaction.
